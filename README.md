# Sort Methods in Java
#### Sorting methods in java for studies

Algorithms studied in class:
- Direct Insertion and Binary Insertion;
- Direct Selection;
- Bubble and Shake;
- Shell;
- Heap;
- Quick (with and without pivot);
- Direct Merge (two implementations).
